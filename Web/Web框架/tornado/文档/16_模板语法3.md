### 模板继承 ###
1. templates/创建base.html文件
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    {%block css%}
    {%end %}
</head>
<body>
    {% block main%}
    {% end%}
</body>
</html>
```
2. templates/创建testextends.html文件
```
{% extends 'base.html' %}
{% block css%}
<style>
    h3{color:red;}
</style>
{% end%}


{%block main%}
{% include 'banner.html'%}
<h3>hello templates</h3>


{%end%}
```

3. py文件中读取
```
#coding=utf-8
import tornado.web
import tornado.ioloop
from tornado.template import Template



"""
模板继承
"""
class IndexHandler(tornado.web.RequestHandler):
    def get(self, *args, **kwargs):
        #在app对象指定模板根目录即可
        self.render('testextends.html')



import os
app = tornado.web.Application([
    (r'/',IndexHandler)


],template_path=os.path.join(os.getcwd(),'templates'))

app.listen(8888)

tornado.ioloop.IOLoop.instance().start()

```
